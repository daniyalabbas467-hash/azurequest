# ☁️ AzureQuest

**A free, gamified study app for Microsoft certification exams — all in a single HTML file.**

### 👉 Use it now: **[tinyurl.com/azurequestapp](https://tinyurl.com/azurequestapp)**

No install, no sign-up, no server. Open the link (or download `index.html` and double-click it) and everything runs in your browser.

## 📚 Covers six exams, A to Z

| Exam | Certification |
|------|---------------|
| ☁️ **AZ-900** | Azure Fundamentals |
| 🚀 **AZ-104** | Azure Administrator Associate |
| 📊 **DP-900** | Azure Data Fundamentals |
| 📈 **PL-300** | Power BI Data Analyst Associate |
| 🤖 **AI-900** | Azure AI Fundamentals |
| 🛡️ **SC-900** | Security, Compliance & Identity Fundamentals |

📱 **Installs as an app**: open the link on your phone → browser menu → *Add to Home Screen*. Works fully offline after the first visit.

## ✨ What's inside

- **Guided lessons** — bite-sized cards with analogies and exam tips, each followed by a quick-check question
- **Practice quizzes** — 500+ questions with instant feedback and explanations, drillable by topic
- **Timed mock exams** — scored to 700/1000 like the real thing, with flag-for-review, answer elimination, a review screen, and per-domain breakdowns
- **Scenario practice** — case-study style linked-question sets, like the real role-based exams
- **Smart review** — missed questions return on a 1 → 3 → 7 day schedule until you master them
- **Cheat sheets** — printable last-mile revision tables per cert
- **Shareable scorecards** — turn a mock result into a PNG for WhatsApp
- **Flashcards & study sessions** — searchable glossary plus a know/again-later review deck
- **Hands-on labs** — step-by-step Azure portal and Power BI Desktop exercises
- **Exam readiness meter** — blends curriculum progress, quiz accuracy, and mock scores to tell you when to book
- **Streaks, XP, levels & badges** — daily goals, a built-in 30-minute study timer, confetti when you earn it
- **Your progress stays yours** — saved locally in your browser, with JSON export/import to move between devices

## 🔗 Official resources linked in-app

Study guides, free practice assessments, Microsoft Learn paths, and the Power BI Desktop download — all one tap away from the Resources tab.
